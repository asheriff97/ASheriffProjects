FILTER BY RECIPE'S INGREDIENT

This page is used to filter recipes by ingredient

For example, if you search for "Bread" you will see all the recipes that contain Bread

If you dont know how many ingredients are available to view, you can check out "All Ingredients" above

Example http://127.0.0.1:5000/api/ingredients?ingredient_name=Bread to filter all recipes that include Bread as an ingredient
