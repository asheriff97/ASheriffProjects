RECIPE API

A prototype API for searching for your favorite home recipes!

Type /api/recipes/all to get a list of all recipes in the database.

Type /api/ingredients/all to get a list of all ingredients in the database.

Type /api/recipes?recipeid=*id* or /api/recipes?recipe_name=*name* to search specific recipes by id or name.

Example http://127.0.0.1:5000/api/ingredients?ingredient_name=Bread to filter all recipes that include Bread as an ingredient
